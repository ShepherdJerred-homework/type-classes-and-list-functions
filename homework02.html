Write the following Haskell functions. You must declare the types for all functions. You should make your types as general as possible; use type classes where constraints are needed.
  1. Write a function named dotProduct which takes two parameters. The parameters should be pairs representing 2-dimensional vectors. You are to calculate the dot product of the two vectors. The dot product of a = (ax, ay) and b = (bx, by) is defined as a · b = axbx + ayby. 
    ghci> dotProduct (2, 6) (3, 1)
12
  2. Write a function named length which takes one parameter. The parameter should be a pair representing a 2-dimensional vector. You are to calculate the length of the vector. The length of a vector v is defined as √v · v. 
    ghci> length (3, 4)
5
  3. Write a function named isUnique which takes a list and returns True if and only if every element in the list occurs only in the list only once 
    ghci> ns = [ 3, 1, 4, 1, 5, 9, 2, 6 ]
ghci> isUnique ns
False
ghci> isUnique (drop 2 ns)
True
  4. Write a function named isAscending which takes a list and returns True if and only if the list is sorted in ascending order 
    ghci> isAscending [1, 2, 3, 5, 8, 13]
True
ghci> isAscending [1, 6, 8, 4, 12, 22]
False
  5. Write a function named to which behaves as the range list construction in Haskell. It should take two arguments and create a list containing all the elements from one to the other.

    You should not assume that the first argument is less than the second argument; if it is greater than, then you should count down. Use the pred and succ functions to count up or down. 

    ghci> 3 `to` 7
[3,4,5,6,7]
ghci> 12 `to` 8
[12,11,10,9,8]
ghci> 'p' `to` 'v'
"pqrstuv"
